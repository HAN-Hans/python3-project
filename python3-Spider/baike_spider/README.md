# baike_spider
get the python entry from baidu-baike

***

## 简介：
**声明:** 定向爬取百度百科python词条下面的所有关于python的链接，并将返回所有页面的标题和简介，最终生成html文件

### 项目环境
**语言：** Python3.6
**系统：** windows10
**编辑器：**  sublime text3

**技术路线**   urllib.request-BeautifulSoup-re

## 四大组件
+ Spider 调度器
+ URL 管理器
+ HTML 下载器
+ HTML 解析器
+ HTML 输出器
